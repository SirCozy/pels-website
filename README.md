# Pel’s Artistry Website

A modern single-page Next.js App Router website for Pel’s Artistry, built with Tailwind CSS. It showcases handcrafted décor products and drives sales through WhatsApp contact.

## Available Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`

## Project structure

- `app/` — App Router pages and global layout
- `components/` — reusable UI components
- `lib/` — editable product data array
- `public/` — static assets and product illustrations

## Notes

- WhatsApp orders use a `wa.me` click-to-chat link
- The design is mobile-first, responsive, and optimized for fast static deployment
